DM panel created by Jordan Taylor

The DM panel allows dungeon masters to play atmospheric music, takes notes, and create NPCs on the fly while DMing



Notes on code below::

	Used express to create the skeleton for this project

	Main code logic is in public/javascripts/global.js (See there for a general description of the project, as well as functions that handle music and character creation)
	
	Main UI code is in views/panelhomepage.jade (coded in Jade)
